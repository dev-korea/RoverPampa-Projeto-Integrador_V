# RoverPampa-Projeto-Integrador_V
O Rover Pampa é um robô móvel de arquitetura distribuída projetado para operar em modos manual e autônomo. Ele utiliza visão computacional para captura de imagens e sensores para navegação e monitoramento ambiental, integrando-se a um aplicativo móvel via Bluetooth Low Energy (BLE) e persistindo dados na nuvem.
